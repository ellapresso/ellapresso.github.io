- Cascading 이란?

- CSS 는?
  - html의 색상이나 크기, 이미지크기, 위치, 배치등 디자인적 요소를 담당.
  - 잘 정리한 css파일의 경우 다른 html또는 템플릿 등에 적용 가능.
  - **자바스크립트**와 연계하여 동적인 디자인 표현이 가능.
  - 동일한 문서내용에 다른 디자인을 입히거나, 동일한 디자인에 다른 문서를 만들기에 용이
  - 반응형 디자인 생성 가능
