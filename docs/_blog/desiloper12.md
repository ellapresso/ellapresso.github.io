##### 16진수로 표시되는 컬러

```css
* {
  color: #aab9ff;
}
```

##### 색 이름 표시

- red, blue, black, violet등의 색 이름으로 표기한다. 

```css
* {
  color: red;
}
```

##### RGB 좌표

- red, green, yellow 함량율을 표기한다.
- 0 ~ 255 사이의 숫자로 표기
- 또는 퍼센트로 표기

```css
* {
  color: rgb(96, 21, 158)
  /* 또는 */
  color: rgb(60%, 21%, 70%)
}
```
