웹 크롤링 도중 발생한 이슈이다.
각 사이트의 이미지 주소를 크롤링하다 보면 이미지 주소의 형태가 각기각색이다.
처음부터 잘 판별하여 긁어오면 좋겠지만,
누군가 작업해 둔 이후 결과물을 다시 유효성 검사를 통해 다듬는 과정에서 삽질한 내역이다.
