#### 노드란?

크롬(Chrome) V8 자바스크립트 엔진으로 빌드된 자바스크립트 **런타임**입니다.

쉽게 자바스크립트 실행기(그런데 빠른) 라고 생각하면 됩니다.

- 서버가 아닙니다.

  - 서버의 역할도 수행
  - 자바스크립트로 작성된 서버를 실행

- 노드가 아니어도 자바스크립트를 실행 할 수 있다.
  - ex) 예전에는 브라우저에서 사용
